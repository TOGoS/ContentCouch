A ContentCouch 'repository' is a directory that stores a bunch of data together:

...repo/urn-res.txt - holds URN resolution information
...repo/heads.rdf - points to latest files/collections
...repo/repos.rdf - information about repositories

URN resolution info is stored as a list of sets of equivalent URIs - each
line stores a tab-separated set of URIs that are equivalent.  Lines beginning
with '#' and blank lines are comments and should be ignored.

# Begin example urn-res.txt
urn:sha1:COOLANDWOOT	http://datastore.nuke24.net/data/COOL/COOLANDWOOT
usn:sha1:WHOAAWESOME	freenet:CHK@g00pfr00t
# End example urn-res.txt

heads.rdf contains the current state of any number of files, directories,
colletcions, or repositories.

<rdf:RDF
	xmlns:rdf="http://www.w3.org/1999/02/22-rdf-syntax-ns#"
	xmlns:dc="http://purl.org/dc/terms/"
	xmlns="http://www.nuke24.net/projects/ContentCouch/ns/"
>
	<Head rdf:ID="togos-favorite-file">
		<dc:title>TOGoS's Favorite File</dc:title>
		<dc:date>2008-12-16 11:21:33</dc:date>
		<type>File</type>
		<content rdf:resource="urn:sha1:COOLANDWOOT"/>
	</Head>

	<Head rdf:ID="togos-photos">
		<dc:title>TOGoS's Photo Collection</dc:title>
		<dc:date>2008-12-16 11:21:39</dc:date>
		<type>Collection</type>
		<listing rdf:resource="usn:sha1:WHOAAWESOME"/>
	</Head>

	<Head rdf:ID="contentcouch">
		<dc:title>ContentCouch</dc:title>
		<dc:date>2008-12-16 11:25:30</dc:date>
		<type>Git-Commit</type>
		<commit rdf:resource="git-commit:0f0f0f0f"/>
	</Head>
</rdf:RDF>

repos.rdf holds information about repositories that are expected to have
information that is referenced by heads.rdf but that cannot be listed in
uri-res.txt.  What information to give about each repository depends on
the respository type.  The general format of repos.rdf is:

<rdf:RDF
	xmlns:rdf="http://www.w3.org/1999/02/22-rdf-syntax-ns#"
	xmlns:dc="http://purl.org/dc/terms/"
	xmlns="http://www.nuke24.net/projects/ContentCouch/ns/"
>
	<rdf:Description about="http://git.nuke24.net/repos/ContentCouch.git/">
		<repositoryType>Git</repositoryType>
		<dc:modified>2008-12-15 12:33:04</dc:modified>
		<gitObjectListing rdf:resource="http://git.nuke24.net/repos/ContentCouch.git/objectlist.txt"/>
	</rdf:Description>
</rdf:RDF>
