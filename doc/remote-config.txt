A remote repository configuration file lists all the remote repositories that you care about.
It is of the following format:

	<option-name> = <value>
	<option-name> ?= <default value>
	
	<tag1>, <tag2>, <tag3>, <unique-name>:
		repo-type <repo-options>

${option-name} anywhere in the file will be replaced by the current value of that option.

e.g.

	default-datastore = F:/datastore/contentcouch/
	
	default,music,music-win:
		cc-collection 
	
	default,local-repo:
		cc-
